# Progressive Web Apps (PWA)

Example `manifest.json` file:

```
{
    "name": "",
    "scope": "/",
    "display": "standalone",
    "start_url": "https://example.com/",
    "short_name": "",
    "theme_color": "transparent",
    "description": "",
    "orientation": "any",
    "background_color": "transparent",
    "related_applications": [],
    "prefer_related_applications": false,
    "icons": []
}
```
